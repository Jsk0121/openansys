OpenAnsys Agent

A local agent workflow that reads engineering input materials, generates ANSYS APDL, repairs failed runs, and prepares LS-DYNA keyword export through a two-stage pipeline.

Pipeline

Built for iterative engineering runs

01

Collect inputs

Scans the local input directory and builds a structured source summary.

02

Generate APDL

Creates Stage 1 ANSYS command streams for geometry, material setup, and meshing.

03

Repair failures

Uses run output and ANSYS error files to drive a focused APDL repair loop.

04

Export K files

Runs Stage 2 workspaces for LS-DYNA-oriented export and validates keyword output.

Local usage

Run the CLI from the repository

Configure your API key and ANSYS executable path, place source materials under input/, then start the interactive agent. 

cargo install --path .
openansys-agent --auto

# Inside the session
/run
/run-stage2
